# 🟦 C++ Core Programming Projects

This repository contains a collection of C++ programs and assignments developed during my first semester of Computer Engineering at the University of Bergamo (UniBG).

The projects focus on core software development principles, clean code, algorithmic logic, and user input validation.

---

## 🛠️ Key Concepts Covered

* **Procedural Programming:** Variables, data types, and control flow (conditionals, loops).
* **Data Robustness:** Comprehensive input validation to handle edge cases and prevent execution errors.
* **Modular Code:** Function structure, proper scoping, and header utilization.

---

## 📂 Featured Applications

### 🌡️ Temperature Converter
* **Description:** A CLI application that converts temperatures between Celsius, Fahrenheit, and Kelvin scales.
* **Highlights:** Includes strict input validation to prevent users from typing characters instead of floating-point numbers.

### 🧮 Menu-Driven Calculator
* **Description:** A mathematical utility that performs operations via a clean, interactive terminal interface.
* **Highlights:** Implements recursive structure choices and safety checks for mathematical errors (such as division by zero).
